# 🏥 Arabic Medical AI Assistant (ArabMedRAG)

> **Production-ready Arabic Medical Chatbot** powered by **CrewAI**, **Hybrid RAG** (FAISS + BM25),
> **Multilingual Sentence Transformers**, and **Qwen** LLM — grounded on 341,476 Arabic medical Q&A pairs.

---

## 📐 Architecture

```
User Query
    │
    ▼
① Language Detection Agent     [langdetect]
    │
    ▼
② Medical Classification Agent [keyword classifier + emergency routing]
    │
    ├──► 🚨 EMERGENCY DETECTED → Immediate Arabic alert (fast path)
    │
    ▼
③ Hybrid Retrieval Agent       [FAISS (SentenceTransformers) + BM25 fallback]
    │
    ▼
④ Citation Grounding Agent     [numbered citations formatter]
    │
    ▼
⑤ Arabic Medical Response Agent [Qwen LLM — Arabic only]
    │
    ▼
⑥ Hallucination Detection Agent [lightweight content-word overlap]
    │
    ▼
Final Arabic Medical Answer (with citations + disclaimer)
```

## 🛠️ Technology Stack

| Component       | Technology                                  |
|-----------------|---------------------------------------------|

## ⚡ Quick Start

### 1. Create conda environment

```bash
conda env create -f environment.yml
conda activate arabic_chatbot
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env — set LLM_PROVIDER (ollama or openrouter)
```

### 3. Pull Qwen model (if using Ollama)

```bash
ollama pull qwen2.5
```

### 4. Build the search indexes

```bash
# Quick test — sample 1,000 rows (~2 min)
python src/medical_chatbot/rag/build_index.py --sample 1000

# Full dataset — 341,476 rows (~20-30 min on CPU)
python src/medical_chatbot/rag/build_index.py
```

### 5. Run the chatbot

**Web UI (recommended):**
```bash
streamlit run app.py
```

**CLI — interactive mode:**
```bash
python src/medical_chatbot/main.py
```

**CLI — single query:**
```bash
python src/medical_chatbot/main.py --query "ما هي أعراض ارتفاع ضغط الدم؟"
```

## 🔍 Hybrid Retrieval Logic

1. **Vector search** (FAISS + SentenceTransformers) always runs first → top-5 results
2. If `max_similarity < 0.50` → **BM25 keyword search** also runs (fallback)
3. Results are merged and deduplicated → top-5 final chunks returned
4. Each chunk includes: category, original question, text snippet, score

## 🚨 Emergency Detection

The classifier agent instantly detects life-threatening symptoms:

- ألم حاد في الصدر / chest pain
- ضيق في التنفس / difficulty breathing
- فقدان الوعي / loss of consciousness
- سكتة دماغية / stroke
- نزيف حاد / severe bleeding

**Response (bypasses full pipeline):**
> "⚠️ قد تشير هذه الأعراض إلى حالة طبية طارئة. يرجى طلب المساعدة الطبية فورًا أو الاتصال بخدمات الطوارئ."
